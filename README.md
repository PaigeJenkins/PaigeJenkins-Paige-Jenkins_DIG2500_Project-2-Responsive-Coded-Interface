# Project Title

## Project Description
This is a single page website, for someone who is looking to explore Ireland.

## Intended User or Player
Any potential tourist of Ireland, to look for destinations, create an itinerary, and drop pins on the map.

## User / Player Goal
The user can like/pin their itinerary, and map out where they would like to go.

## Inspiration Interfaces
I like the different Irish tourism websites, I loved how they incorporated irish scenery, so I did too. I also like AirBnB cards, where you can scroll through cities. So I used that idea to scroll through different destinations in Ireland.

## Design Choices
- Layout: Large hero image, with a greeting text. Followed by destination cards, itinerary list, and a map.
- Color: Green (emerald isle, of course), white, I wish I included more blue.
- Typography: Helvetica, simple, clean.
- Visual hierarchy: Bolded and larger titles, smaller body text
- Feedback states: Pin highlights,
- Responsive design: Nav buttons go to the section, and the cards have a scroll wheel, and pin is clickable.
